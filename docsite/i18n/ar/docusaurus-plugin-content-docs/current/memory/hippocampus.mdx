# عقل ذاكرة Hippocampus

Anda Hippocampus هو نظام الذاكرة طويلة المدى في Anda Bot. ليس قاعدة vectors عادية ولا ملف Markdown لا ينتهي، بل عقل ذاكرة مصمم للوكلاء.

يحول المحادثات والملفات ونتائج الأدوات وسياق القنوات إلى Cognitive Nexus: أشخاص، مشاريع، تفضيلات، أحداث، قرارات، خطوط زمنية وعلاقات.

## حلقة من ثلاث مراحل

### Formation

ترمز المحادثات والنتائج المفيدة إلى ذاكرة منظمة: من، أي مشروع، أي تفضيل، أي قرار ومتى حدث.

### Recall

يسمح بسؤال الرسم المعرفي باللغة الطبيعية. الاسترجاع الجيد يشرح المهمة الحالية بدل إرجاع نص مشابه فقط.

### Maintenance

يدمج الأجزاء، يزيل التكرار، يقلل أثر المعرفة القديمة، ويحفظ الخطوط الزمنية عندما تتغير الحقائق.

## كيف تعلم Anda

```text
تذكر: أفضل ملاحظات release قصيرة مع قسم للمخاطر.
تغير مزود النماذج في هذا workspace من A إلى B.
عندما نذكر Alice فنحن نقصد مصممة فريق الجوال.
```

## ذاكرة جيدة

- سياق واضح: مشروع أو شخص أو ملف أو قناة أو فترة.
- تطور: اذكر النسخة القديمة والجديدة.
- حدود: ضع علامة على المعلومات المؤقتة.
- تصحيح: إذا تذكر Anda خطأ، أعطه النسخة الصحيحة.

## البيانات المحلية والمزودون

يحفظ الرسم في `~/.anda/db` افتراضيًا. قد يشارك مزودو النماذج في formation و recall والتلخيص، لذا اختر endpoints حسب متطلبات الخصوصية.
