# وثائق Anda Bot

Anda Bot مساعد AI محلي يبدأ من الذاكرة. يحافظ desktop launcher و daemon المحلي على Brain داخل جهازك، بينما يمكن للوحة المتصفح والطرفية والملفات و skills و cron و subagents والقنوات والصوت تغذية الذاكرة الرسومية نفسها.

هذه الوثائق لمن يريد امتلاك طبقة المساعد بدلاً من استئجار الذاكرة من حساب نموذج واحد. ابدأ بتثبيت التطبيق، وصل المتصفح، أعد المزودين، وتعلم كيف يحافظ Brain على السياق قابلاً للنقل بين الجلسات والنماذج.

## النموذج الذهني

Anda ليس نافذة دردشة أخرى فقط. الفرق هو الاستمرارية.

- يعمل محليًا ويحفظ الحالة في `~/.anda`.
- يحول Brain السياق المفيد إلى ذاكرة رسومية للمشاريع والتفضيلات والعلاقات والقرارات والجداول الزمنية.
- يمكن تبديل النماذج والمزودين؛ تبقى الذاكرة محلية وقابلة للفحص.
- المتصفح والطرفية والملفات و skills و cron و subagents والقنوات والصوت تتصل بنفس runtime.
- يمكن ل `/goal` الحفاظ على العمل طويل المدى، ويمكن ل `/loop` تشغيل متابعات متكررة أو بإيقاع النموذج.

## أقصر طريق

```bash
brew install ldclabs/tap/anda
DEEPSEEK_API_KEY=YOUR_API_KEY anda
```

إذا لم تكن على macOS أو تفضل launcher أو السكربت، تابع [تثبيت Anda](./quick-start/install).

:::tip
في أول تشغيل، ينشئ Anda ملف `~/.anda/config.yaml`. إذا أبلغت الواجهة عن حقول ناقصة، عدل الملف ثم اضغط Enter لإعادة التحميل.
:::
