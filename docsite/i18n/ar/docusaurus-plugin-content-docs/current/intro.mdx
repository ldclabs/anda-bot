# وثائق Anda Bot

Anda Bot هو مساعد ذكاء اصطناعي محلي قائم على الذاكرة أولاً. يحافظ مشغل سطح المكتب والخدمة الخلفية المحلية (daemon) على تشغيل Brain على جهازك الخاص، بينما يمكن للوحة الجانبية للمتصفح، والطرفية، والملفات، والمهارات (Skills), والمهام المجدولة (cron)، والوكلاء الفرعيين، والقنوات، والوظائف الصوتية تغذية ذاكرة الرسوم البيانية طويلة المدى نفسها.

هذه الوثائق مخصصة للأشخاص المستعدين لامتلاك طبقة المساعد الخاصة بهم بدلاً من استئجار الذاكرة من حساب نموذج واحد. ابدأ بتثبيت التطبيق، ثم قم بتوصيل المتصفح، وتكوين المزودين، وتعرف على كيفية احتفاظ Brain بالسياق قابلاً للنقل عبر الجلسات والنماذج.

## ابدأ هنا

- [تثبيت Anda](./quick-start/install): اختر المشغل، أو Homebrew، أو سكريبت التثبيت، أو Windows PowerShell، أو مسار المصدر.
- [ربط المتصفح](./quick-start/browser-extension): قم بتوصيل اللوحة الجانبية بـ daemon المحلي باستخدام Gateway URL و Bearer token.
- [استخدام الطرفية](./quick-start/terminal): تعرف على إرسال الرسائل، والإدخال متعدد الأسطر، و `/goal`، و `/loop`، و `/side`، و `/skill`، و `/steer` والأوامر الشائعة.
- [تكوين بيئة التشغيل](./runtime/configuration): قم بإعداد `~/.anda/config.yaml` ومزودي النماذج ومفاتيح API والبروكسي والأدلة المحلية.
- [فهم دماغ الذاكرة Brain](./memory/brain): تعرف على حلقة التكوين (Formation) والاستدعاء (Recall) والصيانة (Maintenance).
- [تشغيل المهام طويلة المدى](./workflows/long-horizon): فوض الأهداف المعقدة لأدوار متخصصة مع الحفاظ على المسار الرئيسي والتحقق النهائي.
- [ربط القنوات والصوت](./runtime/channels): اجلب Anda إلى Telegram و WeChat و Discord و Lark/Feishu ومدخلات/مخرجات الصوت.

## النموذج الذهني

Anda Bot ليس مجرد نافذة محادثة أخرى. الاختلاف يكمن في الاستمرارية.

- يعمل Anda محلياً، مع تخزين الحالة الافتراضية في `~/.anda`.
- يقوم Brain بتحويل السياق المفيد إلى ذاكرة رسوم بيانية للمشاريع والتفضيلات والعلاقات والقرارات والجداول الزمنية.
- النماذج والمزودون قابلون للاستبدال؛ وتظل الذاكرة محلية وقابلة للفحص.
- تتصل اللوحة الجانبية للمتصفح، والطرفية، والملفات, والمهارات، والمهام المجدولة، والوكلاء الفرعيون، والقنوات، والصوت ببيئة التشغيل نفسها.
- يمكن لـ `/goal` أن تحافظ على سير المهام طويلة المدى، بينما يمكن لـ `/loop` تشغيل المتابعات المتكررة أو ذات الوتيرة الذاتية.

## الطريق الأقصر

```bash
brew install ldclabs/tap/anda
DEEPSEEK_API_KEY=YOUR_API_KEY anda
```

إذا لم تكن تستخدم نظام macOS، أو كنت تفضل استخدام المشغل أو سكريبت التثبيت، فتابع مع [تثبيت Anda](./quick-start/install).

:::tip
عند التشغيل الأول، يقوم Anda بإنشاء ملف `~/.anda/config.yaml`. إذا أبلغت واجهة مستخدم الطرفية عن فقدان حقول النموذج، فقم بتحرير هذا الملف وحفظه، ثم قم بتحديث النماذج من المشغل أو اللوحة الجانبية للمتصفح، أو قم بتشغيل `anda models reload`.
:::
