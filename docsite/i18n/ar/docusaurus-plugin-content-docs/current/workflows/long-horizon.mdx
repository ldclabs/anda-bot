# العمل طويل المدى والوكلاء الفرعيون

صمم Anda Bot للأهداف المعقدة التي تحتاج استمرارية. يحفظ الهدف، يفحص التقدم، يضغط السياق، ويواصل عند الحاجة.

## استخدام /goal

```text
/goal أكمل release: الاختبارات، changelog، وثائق التثبيت وملخص المخاطر.
```

يفتح `/goal` أو `/loop` حلقة طويلة. عندما يصبح الوكيل الرئيسي متاحًا، يقرر الفاحص هل النتيجة مكتملة أم تحتاج متابعة.

## استخدام /side

ينشئ `/side` أو `/btw` مهمة جانبية للبحث أو المراجعة أو جمع السياق دون قطع المحادثة الرئيسية.

```text
/side تحقق هل يؤثر هذا الاعتماد على macOS arm64.
```

## استخدام /skill

```text
/skill git-assistant اكتب Conventional Commit من diff الحالي.
```

## دور الوكلاء الفرعيين

يمكنهم البحث أو التنفيذ أو المراجعة أو الإشراف. تحتفظ الجلسة الرئيسية بالهدف والحكم النهائي؛ وينجز الوكلاء الفرعيون العمل المركز.

## الأدوات ومساحة العمل

يتضمن Anda الذاكرة والملاحظات و shell والملفات والمهام والمهارات و cron والوكلاء الفرعيين. تستخدم أدوات shell والملفات `~/.anda/workspace` افتراضيًا.

اذكر الهدف ومعايير القبول والحدود بوضوح.

```text
/goal أصلح docs build. لا تعدل Rust. ينتهي العمل عندما ينجح pnpm --dir docsite build.
```
