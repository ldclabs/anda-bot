# Documentación de Anda Bot

Anda Bot es un asistente local con memoria primero. El launcher de escritorio y el daemon local mantienen Brain en tu máquina, mientras el panel lateral del navegador, la terminal, archivos, skills, cron, subagentes, canales y voz pueden alimentar la misma memoria en grafo.

Esta documentación es para quienes quieren poseer la capa de asistente en vez de alquilar memoria a una sola cuenta de modelo. Empieza con la app, conecta el navegador, configura proveedores y aprende cómo Brain mantiene contexto portable entre sesiones y modelos.

## Modelo mental

Anda no es solo otra ventana de chat. Su valor es la continuidad.

- Se ejecuta localmente y guarda estado en `~/.anda`.
- Brain convierte contexto útil en memoria de grafo para proyectos, preferencias, relaciones, decisiones y líneas de tiempo.
- Los modelos y proveedores se pueden reemplazar; la memoria permanece local e inspeccionable.
- El navegador, la terminal, archivos, skills, cron, subagentes, canales y voz se conectan al mismo runtime.
- Con `/goal` se puede mantener trabajo largo; con `/loop`, seguimientos recurrentes o con ritmo del modelo.

## Ruta más corta

```bash
brew install ldclabs/tap/anda
DEEPSEEK_API_KEY=YOUR_API_KEY anda
```

Si no usas macOS o prefieres el launcher o el script, continúa con [Instalar Anda](./quick-start/install).

:::tip
En el primer arranque, Anda crea `~/.anda/config.yaml`. Si la UI indica campos de modelo faltantes, edita el archivo, guárdalo y presiona Enter para recargar.
:::
