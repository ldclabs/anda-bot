# Мозг памяти Hippocampus

Anda Hippocampus — система долгой памяти Anda Bot. Это не простая vector database и не бесконечный Markdown-файл, а память для агентов.

Она превращает беседы, файлы, результаты инструментов и контекст каналов в Cognitive Nexus: людей, проекты, предпочтения, события, решения, временные линии и связи.

## Три фазы

### Formation

Кодирует полезные беседы и результаты в структурированную память: кто, какой проект, какая настройка, какое решение и когда.

### Recall

Позволяет спрашивать граф естественным языком. Хороший recall объясняет текущую задачу, а не просто возвращает похожий текст.

### Maintenance

Консолидирует фрагменты, объединяет дубликаты, ослабляет старое знание и сохраняет временные линии при изменении фактов.

## Как учить Anda

```text
Запомни: я люблю короткие release notes, но с секцией рисков.
Провайдер модели в этом workspace изменился с A на B.
Когда мы говорим Alice, это дизайнер мобильной команды.
```

## Хорошая память

- Контекст: проект, человек, файл, канал или период понятен.
- Эволюция: указаны старая и новая версии.
- Границы: временная информация явно временная.
- Исправление: если Anda ошибся, дайте верную версию.

## Локальные данные и провайдеры

Граф хранится в `~/.anda/db` по умолчанию. Провайдеры моделей могут участвовать в formation, recall и summaries, поэтому выбирайте endpoints по требованиям приватности.
