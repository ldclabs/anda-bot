# Documentation Anda Bot

Anda Bot est un assistant local centré mémoire. Le lanceur desktop et le daemon local gardent Brain sur votre machine, tandis que le panneau navigateur, le terminal, les fichiers, skills, cron, sous-agents, canaux et voix peuvent nourrir la même mémoire graphe.

Cette documentation s'adresse aux personnes qui veulent posséder la couche assistant plutôt que louer la mémoire à un seul compte modèle. Commencez par l app, connectez le navigateur, configurez les providers et découvrez comment Brain garde le contexte portable entre sessions et modèles.

- [Connecter le navigateur](./quick-start/browser-extension) : relier le panneau au daemon local avec Gateway URL et Bearer token.
- [Utiliser le terminal](./quick-start/terminal) : entrée multiligne, `/goal`, `/loop`, `/side`, `/skill`, `/steer` et commandes courantes.
- [Configurer le runtime](./runtime/configuration) : `~/.anda/config.yaml`, fournisseurs, clés API, proxy et dossiers locaux.
- [Comprendre la mémoire](./memory/brain) : Formation, Recall et Maintenance.

- [Canaux et voix](./runtime/channels) : Telegram, WeChat, Discord, Lark/Feishu et voix.

## Modèle mental

Anda n'est pas une simple fenêtre de chat. Sa différence est la continuité.

- Il s'exécute localement et stocke l'état dans `~/.anda`.
- Brain transforme le contexte utile en mémoire graphe pour projets, préférences, relations, décisions et chronologies.
- Les modèles et providers sont remplaçables; la mémoire reste locale et inspectable.
- Navigateur, terminal, fichiers, skills, cron, sous-agents, canaux et voix se connectent au même runtime.
- `/goal` maintient le travail long; `/loop` lance des suivis récurrents ou au rythme du modèle.

## Chemin le plus court

```bash
brew install ldclabs/tap/anda
DEEPSEEK_API_KEY=YOUR_API_KEY anda
```

Si vous n'êtes pas sur macOS ou préférez le lanceur ou le script, continuez avec [Installer Anda](./quick-start/install).

:::tip
Au premier lancement, Anda crée `~/.anda/config.yaml`. Si la UI indique des champs manquants, éditez le fichier puis appuyez sur Enter pour recharger.
:::
