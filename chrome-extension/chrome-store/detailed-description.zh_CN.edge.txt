Anda Bot 在 Microsoft Edge 侧边栏里加入一个 AI 助手，它能读取并操作你正在浏览的网页和标签页——总结当前标签、跨标签页收集信息、填写表单、点击、导航、截图。真正的 AI 跑在你自己的电脑上（一个你自行安装、并用 Gateway URL 和令牌连接的本地配套程序），所以你的提示词和网页内容都由你掌控的软件处理。

它有什么用途

Anda Bot 适合处理普通聊天窗口不擅长的任务：信息散落在多个网页里、需要反复切换标签页、需要结合本机命令行或项目文件、需要持续跟进一个长目标，或者需要把上一次工作的背景记住并带到下一次。你可以让 Anda 理解当前网页，总结资料，比较来源，整理操作步骤，检查网页应用，准备命令，继续之前的目标，或把浏览器里的上下文交给本机智能体处理。

这个扩展本身是 Edge 和本机 Anda 程序之间的桥梁。真正的推理、工具调用、长期记忆和 Skills 能力来自你电脑上运行的 Anda Bot daemon。扩展通过你生成的 Gateway URL 和 Bearer token 连接到本机服务，让你既能在浏览器侧边栏里方便使用，也能保留本地运行时、shell 工具、文件、技能和长期记忆带来的能力。

Anda 还可以通过 Channel 配置接入微信、Telegram、Discord 或飞书等 IM 工具。配置好这些聊天渠道后，只要你控制的电脑上 daemon 持续运行，就可以从远程聊天应用向同一个本机智能体发消息。

为什么应该安装

1. 从正在看的网页开始工作
Anda 可以接收当前标签页的标题和 URL 作为上下文；在你授权并发起相关任务时，它还可以使用浏览器动作，例如列出标签页、打开网页、切换标签页、导航、读取页面、填写输入框、点击元素、滚动和截图。这让它很适合资料研究、网页内容整理、表单类流程、网页应用检查，以及任何“浏览器就是工作现场”的任务。

2. 把浏览器和命令行连起来
很多电脑工作不只发生在网页里。Anda 的优势在于把网页上下文和本机工具连接起来：命令行程序、项目文件、日志、脚本、开发工具和自动化流程都可以成为同一个任务的一部分。扩展提供了 Edge 侧边栏入口，而背后连接的是同一个会使用终端工具的本机智能体。

3. 推进长程任务
Anda Bot 面向长程推理任务，而不是只给出一次性回答。一个目标可以保持活跃：Anda 会压缩上下文、保留目标、调用工具、协调子智能体，并持续推进到有证据表明任务已经完成。这对资料调研、问题排查、多步骤配置、文档准备和周期性运营工作尤其有用。

4. 通过 Skills 扩展能力
Skills 可以让 Anda 学会更专门的工作流。你可以为特定领域、文件格式、工具或团队流程添加 Skills，并从同一个 Edge 对话里使用它们。随着 Skills 增加，扩展不只是一个通用聊天窗口，而会逐渐变成贴近你真实工作方式的本地协作者。

5. 基于图谱记忆大脑持续学习
Anda Hippocampus 会把有价值的对话转化为图谱长期记忆：项目、人物、偏好、决策、事件、关系和时间线。它的目标不是简单保存聊天记录，而是让 Anda 在未来任务里召回真正有用的背景，理解变化，逐渐适应你的工作方式。

6. 通过 IM 工具远程使用 Anda
Edge 很适合处理网页现场的任务，但你也可能希望离开浏览器后继续向 Anda 提问。Anda 支持微信、Telegram 等 IM 和团队聊天渠道接入；配置完成后，消息会到达同一个长期运行的本机 daemon，让 Anda 可以从远程聊天界面继续帮你处理提醒、调研、追问和运营类任务。

安装前需要知道

Edge 扩展需要配合本机 Anda Bot 程序使用。安装扩展后，请先在电脑上安装 Anda，配置至少一个模型服务商 API Key，用后台模式启动 daemon，然后生成扩展令牌：

anda start

anda browser token --days 365

把命令输出的 Gateway URL 和 Bearer token 粘贴到扩展设置中。连接成功后，就可以在 Edge 侧边栏里与 Anda 对话。

完整安装指南：https://docs.anda.bot/docs/quick-start/browser-extension

隐私和权限说明

扩展会把 Gateway URL、Bearer token 和浏览器 session id 保存在 Edge 本地存储中。它会把你的提示词、你选择的附件、浏览器 session 元数据，以及相关标签页信息发送给你连接的本机 Anda daemon。为了完成浏览器任务，扩展需要访问标签页、当前页面、截图、页面脚本、文本转语音，以及可选的语音录制能力。你本机的 Anda 运行时，以及你自己配置的模型服务商，可能会处理你的 prompt 和任务数据。发布或在敏感场景中使用前，请同时查看隐私政策。

扩展还会使用 downloads 权限处理浏览器发起的文件下载，使用 cookies 权限在用户请求时查看和更新 Cookie，使用 browsingData 权限在重置页面状态时清理缓存和相关站点数据，使用 debugger 权限更稳定地执行页面自动化并生成截图或导出 PDF 等结果，以及使用 webNavigation 权限感知页面跳转和加载完成状态，让智能体可以等待标签页真正就绪。

适合谁使用

Anda Bot 面向所有需要在电脑上浏览、写作、研究、排查问题、整理资料和推进任务的人。你不需要是开发者也可以使用它；但如果你希望一个助手既能理解浏览器上下文，又能使用命令行工具、记住重要背景，并持续推进需要时间的目标，它会特别有价值。
